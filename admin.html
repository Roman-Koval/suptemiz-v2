⚠️ Нет прав доступа к Firestore. Откройте Firebase Console → Firestore Database → Правила и установите (только для разработки!): 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
Для продакшна — добавьте аутентификацию и ограничьте доступ.
